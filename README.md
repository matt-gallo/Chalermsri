# Chalermsri Coffee Roaster · บ้านเฉลิมศรี

Static one-page site for Chalermsri Coffee Roaster — a family-run hand-roastery
at the Phra That Doi Saket junction in Chiang Mai.

## Run locally

It's pure HTML/CSS/JS — open `index.html` in a browser, or:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Files

- `index.html` — markup, sections (hero, story, menu, reviews, visit, contact)
- `styles.css` — styling
- `script.js` — bilingual EN/TH toggle + contact form (mailto)
